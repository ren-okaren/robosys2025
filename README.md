# robosys2025

ロボットシステム学の練習用リポジトリです。  
標準入力から数値や文字列を読み込んで処理するコマンドなどを作っています。

## 必要なソフトウェア

- Python 3
- Bash
- Git

## テスト環境

- Ubuntu 22.04 LTS
- GitHub Actions (`ubuntu-latest`)

## インストール・準備

以下のコマンドを実行してリポジトリを取得し、実行権限を付けます。

~~~sh
git clone https://github.com/ren-okaren/robosys2025.git
cd robosys2025
chmod +x plus macprice
~~~

---

## plus コマンド

標準入力から読み込んだ数値をすべて足し合わせて、合計を標準出力に出すコマンドです。

### 実行例

~~~sh
$ seq 5 | ./plus
15
~~~

### 入力

- 数値を 1 行に 1 つずつ入力
- 入力終了は `Ctrl + D`（EOF）

### 出力

- 合計値を 1 行で出力

---

## macprice コマンド

マクドナルドのメニュー名をカンマ区切りで標準入力から読み込み、  
各商品の合計金額を標準出力に表示する Python 製コマンドです。

### 実行例

~~~sh
$ printf "ビッグマック,サイドサラダ,ビッグマック,コカ・コーラ\n" | ./macprice
1430
~~~

### 入力

- カンマ区切りでメニュー名（日本語名）を 1 行で入力  
  例）`ビッグマック,サイドサラダ,ビッグマック,コカ・コーラ`
- 同じ商品名が複数回現れた場合は、その回数を個数として数えます。
- 入力の終了は `Ctrl + D`（EOF）で指定します。

### 出力

- 合計金額を 1 行で出力  
  例）`1430`

### エラー処理

次の場合はエラーメッセージを標準エラー出力（stderr）に表示し、終了ステータス 1 で終了します。

- 未登録のメニュー名が入力された場合
- 有効な注文が 1 つも入力されなかった場合
- 行頭に個数だけが入力されていて商品名が空の行だった場合

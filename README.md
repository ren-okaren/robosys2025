# robosys2025

ロボットシステム学の練習用リポジトリです。  
標準入力から数値や文字列を読み込んで処理するコマンドなどを作っています。

## インストール・準備

次のコマンドでリポジトリを取得し、実行権限を付けます。

    git clone https://github.com/ren-okaren/robosys2025.git
    cd robosys2025
    chmod +x plus macprice

## plus コマンド

標準入力から読み込んだ数値をすべて足し合わせて、その合計を標準出力に出すコマンドです。

### 実行例

    $ seq 5 | ./plus
    15

### 入力

- 数値を 1 行に 1 つ  
- 入力の終了は `Ctrl + D`（UNIX 系環境での EOF）で指定します。

### 出力

- 合計値を 1 行で出力

## macprice コマンド

マクドナルドのレギュラーメニュー名を標準入力から読み込み、  
各商品の値段と合計金額を標準出力に表示するプログラムです。
起動時に一度、登録されているメニュー一覧を表示します。

### 実行例

    $ printf "2ビッグマック\nサイドサラダ\nコカ・コーラ\n" | ./macprice
    === メニュー一覧 ===
    …（メニュー一覧が表示される）…
    ビッグマック 2個 960円
    サイドサラダ 1個 350円
    コカ・コーラ 1個 140円
    合計金額は 1450 円になります

### 入力

- メニュー名（日本語名）を1行に1つ
- 個数を指定する場合は、行頭に数字を書きます。
  - 例: `2サイドサラダ` や `2 サイドサラダ`
- 入力の終了は `Ctrl + D` で指定します。

### 出力

- 各行について 
  - `メニュー名 個数 個別の合計金額` を1行で表示
- 最後に 
  - `合計金額は ○○ 円になります` を1行で表示

### エラー処理

次のような不正な入力があった場合はエラーとして終了します。

- 未登録のメニュー名が入力された場合
- 個数が 1 未満、または整数でない場合
- 行頭の個数だけが入力されていて商品名が空の行

これらの場合は、エラーメッセージを標準エラー出力に表示し、
終了ステータス1で終了します。
